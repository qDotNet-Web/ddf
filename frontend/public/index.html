Wie wird gespielt?

Das Spiel besteht aus 3 Runden. In jeder Runde wird eine Frage gestellt, die von den Spielern beantwortet werden muss. Die Spieler müssen die Antwortmöglichkeit auswählen, die sie für richtig halten. Nachdem alle Spieler ihre Antwort abgegeben haben, wird die richtige Antwort angezeigt. Die Spieler, die die richtige Antwort gewählt haben, erhalten Punkte. Der Spieler mit den meisten Punkten nach 3 Runden gewinnt das Spiel.
